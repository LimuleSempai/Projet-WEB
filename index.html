Bienvenue sur notre outil d'analyse de données CSV

Ce site vous permettra d'importer un fichier .csv de votre choix afin d'obtenir une analyse rapide de celui-ci.

Cela vous aidera à orienter vos décisions marketing ou de recherche dans votre domaine ou même encore de faire ressortir les points que vous avez besoin d'analyser plus en profondeur.

L'Overview : Page donnant une première vue globale des données de votre .csv.

L'overview va vous donner un résumé de vos données pour vous aider à mieux comprendre ce que vous voulez manipuler.

Assurez-vous que le fichier dont vous disposez s'agit bien d'un document adequat : mon_fichier.csv.

Vous pouvez alors commencer votre analyse en vous deplaçant sur la page Overview ici ou en utilisant la barre de navigation en haut de la page.

Le Dashboard : Page générant des analyses à partir des données que vous fournissez.

Le dashboard vous donnera une analyse générale de vos données.

L'objectif est de pouvoir vous permettre de prendre vos première décisions par rapport à votre recherche, que votre besoin soit de donner un simple coup d'oeil à vos données ou d'orienter vos choix dans votre domaine.

Sur cette page, le site vous offrira une analyse des données de votre fichier .csv.

Il vous est conseillé d'importer en premier lieu votre fichier sur la page Overview pour obtenir un résumé de vos données afin de mieux les comprendre avant de commencer l'analyse.

Sinon, vous vouvez acceder immediatement à la page Dashboard en cliquant ici ou en utilisant la barre de navigation en haut de la page.
